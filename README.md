Сайт мини-игр
Это веб-сайт с коллекцией из шести простых мини-игр для развлечения и тренировки логики. Сайт предназначен для пользователей любого возраста — от детей до взрослых. Посетители могут выбрать одну из игр и сразу начать играть прямо в браузере. Цель сайта — предоставить лёгкий способ развлечься, потренировать мышление и провести время с пользой. Игры не требуют установки — всё работает в браузере. Список игр: «Угадай число» — нужно угадать случайное число в заданном диапазоне; «Простая арифметика» — решение базовых математических примеров; «Переверни текст» — игра на перестановку букв для получения осмысленного слова; «Камень, ножницы, бумага» — классическая игра против компьютера; «Простая викторина» — ответы на вопросы с вариантами ответов; «Генератор случайных цветов» — игра, меняющая фон страницы при клике.
HTML5 — для структуры страниц; CSS3 — для стилизации и адаптивного дизайна; JavaScript — для логики игр и взаимодействия с пользователем; Google Fonts — для подключения шрифта Montserrat; Git — для контроля версий; Изображения в формате PNG — для оформления карточек игр и декоративных элементов; Семантическая разметка — для улучшения доступности и SEO.
